# Roadmap

## v1.1 — Touch & mobile
- [ ] Touch event support for dragging on mobile (`touchstart`, `touchmove`, `touchend`)
- [ ] Responsive column width (currently fixed at 850 px)
- [ ] Pinch-to-resize the illumination image

## v1.2 — Multiple images
- [ ] Support more than one illumination on the same page
- [ ] Z-index management UI (bring to front / send to back)

## v1.3 — Export
- [ ] "Save as PNG" — flatten canvas + text into a single image
- [ ] "Print" CSS stylesheet with correct page dimensions

## v1.4 — Custom text
- [ ] Textarea in the toolbar to replace the hardcoded source text
- [ ] Markdown subset parser (bold, italic, drop cap)

## v2.0 — Multi-column & page layout
- [ ] Two-column mode mirroring classic manuscript layout
- [ ] Page breaks and folio numbering
- [ ] Configurable margins and column gutter

## Ideas / icebox
- WebGL-accelerated hitmap sampling for very large images
- Collaborative editing via WebRTC (two scribes, one manuscript)
- Drop-cap / versale support (oversized first letter)
